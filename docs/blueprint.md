# **App Name**: ETL Insights Analyzer

## Core Features:

- Log File Upload: Allows users to upload ETL log files in CSV or JSON format.
- Log Parsing and Summarization: Automatically parses uploaded log files and summarizes key metrics like number of errors, successes, and warnings.
- Trend Visualization: Generates visual charts (bar and pie charts) to represent trends in the log data, such as error counts per source system and success vs. failure rates.
- Data Filtering: Enables users to filter log data by date, source system, and status to isolate specific issues.
- Insight Export: Allows users to export filtered log data and summary insights as CSV files for further analysis.

## Style Guidelines:

- Primary color: A calm and reliable blue (#4681C4) to evoke trust and precision in data analysis.
- Background color: Light blue (#D8E2EC) for a clean and unobtrusive backdrop, maintaining focus on the data.
- Accent color: A contrasting orange (#C46E46) to draw attention to key interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean, modern, readable style appropriate to dense information.
- Code font: 'Source Code Pro' for displaying CSV/JSON data.
- Simple, clear icons to represent different data types and actions.
- Clean, organized layout with clear data presentation to support the fast parsing of ETL data.
